DockerFiles - Comandos

Arquivo: Dockerfile
Para gerar uma imagem a partir de Dockerfile: docker build -t nome_imagem.

Comandos Dockerfile:

FROM - Chamada da imagem que será base para a construção da minha imagem.

RUN - Permitir a execução de comandos na montagem da imagem.

CMD x ENTRYPOINT - Executam um determinado comando na inicialização do container gerado a partir da imagem customizada por Dockerfile.

COPY x ADD - Permitem a cópia de 1 ou mais arquivos para uma pasta específica da imagem.

ADD - Serve para copiar arquivos compactados (tar), descompactando na cópia.

EXPOSE - Informa qual porta deverá ser liberada.

VOLUME - Determina o ponto de montagem para uma pasta que ficará disponível entre host e container.

WORKDIR - Determina o diretório de trabalho. Na criação do container a partir da imagem customizada, passa a ser o diretório padrão.

USER - Define o usuário que executa comandos.

LABEL - Define o mantenedor da imagem.
